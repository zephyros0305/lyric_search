# lyric_search 歌詞搜尋器
  如同其名，就是做來搜尋歌詞用的啦！

  主要是爬[魔鏡歌詞網](http://mojim.com/twznew.htm)裡的歌詞來呈現。

  當初想做的原因是想要簡潔的顯現歌詞(~~其實是覺得該網站很多廣告又有廣告詞很煩XD~~)。

  現在免強算是半成品，因為實在是太過簡潔XD!?

  不過我也不一定會再修改它，哈哈
